LET'S LEARN TOGETHER

Simulación de un sistema microcontrolado


Este documento describe el proceso y las herramientas para el diseño y la simulación de un sistema con microcontroladores. Se usa como simulador el programa de diseño electrónico Proteus 8, el cual se supone ya instalado en este momento.

Para la escritura del código se usa el IDE de Arduino preferiblemente la última versión disponible. Para este ejercicio se usa la versión 2.3.x.
